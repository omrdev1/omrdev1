## Open Source Architecture

Systems engineering for high-availability environments.

- **[Saga Engine](https://github.com/grafikui/saga_engine)**  
  Crash-resilient workflow executor for Node.js, Postgres-backed persistence, best-effort compensation (undo) logic, advisory locks for distributed consistency.

- **[Saga Engine Go](https://github.com/grafikui/saga-engine-go)**  
  High-concurrency implementation of the Saga Pattern in Golang, optimized for microservice coordination and atomic transaction rollbacks.
